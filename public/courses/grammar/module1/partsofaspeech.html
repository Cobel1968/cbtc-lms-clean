Session Objective

In this session, learners will understand the four core parts of speech: Nouns, Verbs, Adjectives, and Adverbs. Each concept is presented bilingually (English ↔ French) to reduce cognitive friction and ensure technical clarity.

1. Nouns ↔ Noms

English

Noun: A name of a person, place, thing, or idea.

  • Person: teacher, doctor
  • Place: school, city
  • Thing: book, table
  • Idea: love, freedom
Example: The student reads a book.

Français

Nom : Nom d’une personne, d’un lieu, d’une chose ou d’une idée.

  • Personne : professeur, médecin
  • Lieu : école, ville
  • Chose : livre, table
  • Idée : amour, liberté
Exemple : L’élève lit un livre.

2. Verbs ↔ Verbes

English

Verb: Shows an action or a state of being.

  • Action: run, eat, work
  • State: be, seem, exist
Example: She works every day.

Français

Verbe : Exprime une action ou un état.

  • Action : courir, manger, travailler
  • État : être, paraître, exister
Exemple : Elle travaille tous les jours.

3. Adjectives ↔ Adjectifs

English

Adjective: Describes a noun.

Example: A big house

Français

Adjectif : Décrit ou précise un nom.

Exemple : Une grande maison

4. Adverbs ↔ Adverbes

English

Adverb: Describes a verb, adjective, or another adverb.

Example: He speaks clearly.

Français

Adverbe : Modifie un verbe, un adjectif ou un autre adverbe.

Exemple : Il parle clairement.
