Lesson 1A: "What's Wrong?" - Describing Problems
Grammar Focus: There is / There are
Level: Beginner (IELTS < 4.5)
For: French-speaking Students

Lesson Objective

By the end of this lesson, you will be able to describe maintenance problems in an office using "There is" and "There are" correctly.

Pour nos étudiants francophones: En français, on utilise "Il y a" pour "There is/There are". Cette leçon vous aide à utiliser la forme correcte en anglais (singulier/pluriel).

Grammar Rule: There is / There are

Singular (Une chose) → Use "There is"

There is a broken chair.
Il y a une chaise cassée.
There is a problem.
Il y a un problème.
There is a leak in the bathroom.
Il y a une fuite dans la salle de bain.

Plural (Plusieurs choses) → Use "There are"

There are three broken chairs.
Il y a trois chaises cassées.
There are some problems.
Il y a quelques problèmes.
There are old computers in the office.
Il y a des vieux ordinateurs dans le bureau.

Remember!

  • Use "a" or "an" with singular nouns: There is a broken window.
  • Use numbers or "some" with plural nouns: There are two broken windows.
  • With plural nouns, add "s" to the noun: chair → chairs

Real Practice for Your Role-play

Imagine you are the Maintenance Manager. You need to tell the Finance Manager about problems in the office. Write 5 sentences using "There is" or "There are".

Example list for your role-play:

  1. There is a broken air conditioner in the main office.
  2. There are three old printers.
  3. There is a leak in the bathroom.
  4. There are five broken chairs in the conference room.
  5. There is a problem with the security door.
Pour votre rôle: Préparez votre propre liste en français, puis traduisez en utilisant "There is/There are".
Exemple: "Il y a une fuite dans le couloir" → "There is a leak in the hallway."
